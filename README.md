# sql2xml
Cette commande, écrit en Pascal (Turbo Pascal), permet de convertir un fichier SQL en XML.

<h3>Syntaxe</h3>

<b>SQL2XML</b> source.SQL

<h3>Paramètres</h3>

<table>
  <tr>
    <th>Nom</th>
    <th>Description</th>
  </tr>
  <tr>
    <td><i>source</i></td>
    <td>Ce paramètre permet d'indiquer le fichier de code source SQL.</td>
  </tr>
</table>
